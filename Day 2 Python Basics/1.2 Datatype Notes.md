# Data Types

## What is a Data Type?
A **data type** is a classification that specifies what kind of value a variable can hold in Python. It determines:
- What operations can be performed on the data
- How the data is stored in memory
- The default behavior of the variable

Python is **dynamically typed** — you don't need to declare the type explicitly; Python automatically detects it.

```python
x = 10      # x is int
x = "Hello" # now x is str (same variable can change type)
```

### Why Do We Need Data Types?
- To store values correctly and efficiently
- To perform correct operations (e.g., you can't add a string and integer directly without conversion)
- Memory optimization (int uses less memory than float when not needed)
- Code readability and error prevention

### When Do We Explicitly Use/Convert Data Types?
We use data type functions mainly during type casting (type conversion) when:
- Taking user input (`input()` always returns string)
- Performing operations between different types
- Reading/writing data from files, APIs, etc.

### Common Type Conversion Functions
```
int()    → converts to integer
float()  → converts to float
str()    → converts to string
bool()   → converts to boolean
list()   → converts to list
tuple()  → converts to tuple
dict()   → converts to dictionary
```

## Built-in Data Types in Python

### 1. Integer (`int`)
- Whole numbers (positive, negative, zero)
- No limit on size in Python (unlimited precision)
```python
age = 25
marks = -95
big_num = 999999999999999999999
```

### 2. Float (`float`)
- Decimal/point numbers
- Used for precision calculations
```python
price = 99.99
height = 5.11
pi = 3.14159
```

### 3. String (`str`)
- Sequence of characters enclosed in quotes
- Immutable (cannot be changed after creation)
```python
name = "Alice"
message = 'Hello World'
multiline = """This is\na multiline string"""
```

### 4. Boolean (`bool`)
- Represents truth values: True or False
- Used in conditions and loops
- Note: Always write with capital T/F (coding standard)
```python
is_student = True
has_passed = False
```
Bonus: These values are treated as bool → False, 0, "", [], {}, None are False, everything else is True

### 5. List (`list`)
- Ordered, mutable, allows duplicates
- Can store mixed data types
```python
fruits = ["apple", "banana", "cherry"]
numbers = [1, 2, 3, 4, 5]
mixed = [1, "hello", True, 3.14]
```
Common methods: `.append()`, `.remove()`, `.pop()`, `.sort()`, `len()`

### 6. Tuple (`tuple`)
- Ordered, immutable, allows duplicates
- Faster than lists
```python
coordinates = (10, 20)
colors = ("red", "green", "blue")
```

### 7. Dictionary (`dict`)
- Stores data in key-value pairs
- Keys must be unique and immutable (str, int, tuple)
- Values can be anything
```python
student = {
    "name": "Rahul",
    "age": 20,
    "grade": "A"
}
```
Access: `student["name"]` → "Rahul"
Common methods: `.keys()`, `.values()`, `.items()`, `.get()`

### 8. Set (`set`)
- Unordered, mutable, no duplicates
- Used for membership testing and removing duplicates
```python
unique_nums = {1, 2, 3, 3, 4}  # → {1, 2, 3, 4}
vowels = {"a", "e", "i", "o", "u"}
```
Operations: union (`|`), intersection (`&`), difference (`-`)

### 9. NoneType (`None`)
- Represents absence of value
- Used to initialize variables
```python
result = None
```

## Taking User Input & Output
**Input → `input()`**
- Always returns a string
- Syntax: `variable = input("prompt")`
```python
name = input("Enter your name: ")        # returns string
age = int(input("Enter your age: "))     # convert to int
height = float(input("Height in cm: "))  # convert to float
```

**Output → `print()`**
```python
print("Hello", name)
print("You are", age, "years old")
print(f"Your name is {name} and age is {age}")  # f-string (recommended)
```

## How to Check Data Type?
Use the `type()` function:
```python
x = 100
print(type(x))        # <class 'int'>

y = "Python"
print(type(y))        # <class 'str'>
```

## How to Check if Variable is Certain Type?
Use `isinstance()`
```python
isinstance(10, int)        # True
isinstance("hello", str)   # True
isinstance(3.14, (int, float))  # True (checks multiple types)
```
