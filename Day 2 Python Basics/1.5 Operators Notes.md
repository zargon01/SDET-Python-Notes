# Operators

Operators are special symbols that perform operations on variables and values.

Python has 7 major types of operators:

---

### 1. Arithmetic Operators (Mathematical Operations)

| Operator | Name              | Example          | Result        | Description                          |
|----------|-------------------|------------------|---------------|--------------------------------------|
| `+`      | Addition          | `10 + 5`         | 15            |                                      |
| `-`      | Subtraction       | `10 - 5`         | 5             |                                      |
| `*`      | Multiplication    | `10 * 3`         | 30            |                                      |
| `/`      | Division          | `10 / 3`         | 3.333...      | Always returns **float**             |
| `//`     | Floor Division    | `10 // 3`        | 3             | Returns integer (drops decimal)      |
| `%`      | Modulo (Remainder)| `10 % 3`         | 1             | Very useful for odd/even check       |
| `**`     | Exponent (Power)  | `2 ** 3`         | 8             | 2³ = 8                               |

```python
print(10 // 3)   # 3
print(10 / 3)    # 3.3333333333333335
print(2 ** 10)   # 1024
```

---

### 2. Comparison Operators (Return True/False)
Used in conditions (if, while, etc.)

| Operator | Meaning                  | Example       | Result |
|----------|--------------------------|--------------|--------|
| `==`     | Equal to                | `10 == 10`   | True   |
| `!=`     | Not equal to            | `10 != 20`   | True   |
| `<`      | Less than               | `10 < 20`    | True   |
| `>`      | Greater than            | `10 > 20`    | False  |
| `<=`     | Less than or equal      | `10 <= 10`   | True   |
| `>=`     | Greater than or equal   | `20 >= 10`   | True   |

```python
print(5 == "5")    # False (different types)
print(5 == 5)      # True
```

---

### 3. Assignment Operators (Shortcut for assigning values)

| Operator | Example    | Same As       | Use Case          |
|----------|-----------|---------------|-------------------|
| `=`      | `x = 10`  | Normal assign |                   |
| `+=`     | `x += 5`  | `x = x + 5`   | Increment         |
| `-=`     | `x -= 3`  | `x = x - 3`   | Decrement         |
| `*=`     | `x *= 2`  | `x = x * 2`   | Multiply assign   |
| `/=`     | `x /= 4`  | `x = x / 4`   | Divide assign     |
| `//=`    | `x //= 3` | `x = x // 3`  | Floor division    |
| `%=`     | `x %= 3`  | `x = x % 3`   | Remainder assign  |
| `**=`    | `x **= 2` | `x = x ** 2`  | Power assign      |

```python
score = 90
score += 10   # score becomes 100
print(score)  # 100
```

---

### 4. Logical Operators (Combine conditions)

| Operator | Description                     | Example                | Result |
|----------|---------------------------------|------------------------|--------|
| `and`    | True only if both are True     | `True and False`      | False  |
| `or`     | True if at least one is True   | `True or False`       | True   |
| `not`    | Reverses the result            | `not True`            | False  |

```python
age = 20
print(age >= 18 and age <= 60)   # True
print(age < 10 or age > 60)      # False
print(not (age > 18))            # False
```

**Truth Table Shortcut:**
- `0`, `False`, `""`, `[]`, `{}`, `None` → treated as False
- Everything else → True

---

### 5. Identity Operators (Check memory location)
Compares object identity (not value)

| Operator | Meaning                          | Example       |
|----------|---------------------------------|--------------|
| `is`     | True if same object in memory   | `a is b`     |
| `is not` | True if NOT same object         | `a is not b` |

**Important Difference: `==` vs `is`**

```python
a = [1, 2, 3]
b = [1, 2, 3]
c = a

print(a == b)      # True  → values are same
print(a is b)      # False → different objects
print(a is c)      # True  → same object (c points to a)
```

Use `is` only with `None`, not for general equality!

```python
x = None
if x is None:
    print("x has no value")
```

---

### 6. Membership Operators (Check presence in sequence)
Works with strings, lists, tuples, sets, dictionaries

| Operator   | Meaning                              | Example                          |
|------------|--------------------------------------|---------------------------------|
| `in`       | True if value is found              | `"app" in "apple"` → True      |
| `not in`   | True if value is NOT found          | `"xyz" not in "apple"` → True  |

```python
fruits = ["apple", "banana", "mango"]

print("banana" in fruits)        # True
print("orange" not in fruits)    # True
print("app" in "apple")          # True (substring check)
```

---

### 7. Bitwise Operators (Work on binary bits)
Converts numbers to binary and operates bit by bit.

| Operator | Name        | Example      | Binary Example (5=101, 3=011) |
|----------|------------|-------------|--------------------------------|
| `&`      | AND        | `5 & 3`     | 101 & 011 = 001 → 1           |
| `|`      | OR         | `5 | 3`     | 101 | 011 = 111 → 7           |
| `^`      | XOR        | `5 ^ 3`     | 101 ^ 011 = 110 → 6           |
| `~`      | NOT        | `~5`        | flips all bits → -6           |
| `<<`     | Left Shift | `5 << 1`    | 101 << 1 = 1010 → 10 (×2)     |
| `>>`     | Right Shift| `5 >> 1`    | 101 >> 1 = 010 → 2 (÷2)       |

```python
print(5 & 3)   # 1
print(5 | 3)   # 7
print(5 << 2)  # 20 (5 × 4)
```

---

### Operator Precedence (Order of execution)
From highest to lowest:

```
**
~ (bitwise not)
*, /, //, %
+, -
<<, >>
&
^, |
Comparison (==, !=, etc.)
Assignment (=, +=, etc.)
is, in
not
and
or
```

Use parentheses `()` to control order!
