# Input and Output (I/O)

## OUTPUT in Python → Using `print()`

### 1. Basic Print Statement
You can print anything inside single or double quotes (both work the same).

```python
print('Hello World')
print("Hello World")
print(100, 3.14, True)    # Multiple values with space by default
```

### 2. Dynamic / Formatted Printing (Most Important!)

| Method                | Syntax Example                  | Output Example (name = "Golu") | Notes                                      |
|-----------------------|---------------------------------|---------------------------------|-------------------------------------------|
| String Concatenation  | `print("Hi " + name)`          | Hi Golu                         | Only works with strings → need conversion |
| Comma Separation      | `print("Hi", name)`            | Hi Golu                         | Automatically adds space                  |
| f-strings (Best)      | `print(f"Hi {name}")`          | Hi Golu                         | Recommended in Python 3.6+                |
| `.format()` method    | `print("Hi {}".format(name))` | Hi Golu                         | Older style, still works                  |

**Best Practice → Always use f-strings!**

```python
name = "Golu"
age = 15
print(f"My name is {name} and I am {age} years old")
# Output: My name is Golu and I am 15 years old
```

### 3. The `end` Parameter (Change what comes at the end)
By default, `print()` adds a newline (`
`) at the end.

```python
print("Hello")    # → goes to next line automatically
print("World")
# Output:
# Hello
# World
```

Change it using `end=` → Very useful for patterns!

```python
print("Hello", end=" ")    # space instead of newline
print("World")
# Output: Hello World

print("★", end="")         # no space
print("★", end="")
print("★")
# Output: ★★★
```

### 4. The `sep` Parameter (Separator between values)
By default, comma-separated values are separated by a space.

```python
print("Jan", "Feb", "Mar")              # sep=" " (default)
# Output: Jan Feb Mar

print("Jan", "Feb", "Mar", sep="-")
# Output: Jan-Feb-Mar

print("01", "01", "2025", sep="/")
# Output: 01/01/2025
```

### 5. Formatting Numbers (Decimal Places)

```python
num = 10.345678

print(f"Value = {num:.2f}")   # 2 decimal places
# Output: Value = 10.35 (rounds off)

print(f"Value = {num:.1f}")   # 1 decimal place
# Output: Value = 10.3

print(f"Percentage: {95.5:.0f}%")  # No decimal
# Output: Percentage: 96%
```

Other formatting examples:

```python
price = 499.99
print(f"Price: ₹{price:,.2f}")   # Indian style comma
# Output: Price: ₹499.99
```

## INPUT in Python → Using `input()`

**Key Point:** `input()` ALWAYS returns a string!

```python
name = input("Enter your name: ")
print(f"Hi {name}")
# Even if user types 25 → it's stored as "25" (string)
```

### Taking Different Types of Input

```python
# String input (no conversion needed)
name = input("Enter name: ")

# Integer input
age = int(input("Enter age: "))

# Float input
height = float(input("Enter height in cm: "))

# Multiple inputs in one line
x, y = input("Enter two numbers: ").split()
x = int(x)
y = int(y)

# Or in one line (advanced)
a, b = map(int, input("Enter a and b: ").split())
```

### Safe Input (Avoid Errors)

```python
try:
    age = int(input("Enter age: "))
except ValueError:
    print("Please enter a valid number!")
    age = 0
```
