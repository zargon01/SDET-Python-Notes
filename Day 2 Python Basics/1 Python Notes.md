# Python

### What is Python?

**Python** is a **high-level, interpreted, dynamically-typed** programming language created by **Guido van Rossum** in 1991.

**Simple Definition (Best for Exams):**  
**"Python is a beginner-friendly, high-level programming language that uses simple English-like syntax and is interpreted line by line."**

### Key Features of Python

| Feature                      | Meaning                                                                 |
|--------------------------------|-------------------------------------------------------------------------|
| **High-Level Language**        | Close to human language, easy to read and write                         |
| **Interpreted**                | Code runs line by line (no need to compile)                             |
| **Dynamically Typed**          | No need to declare variable types (e.g., int, string) – decided at runtime |
| **Beginner-Friendly**          | Syntax looks like simple English                                        |
| **Cross-Platform**             | Runs on Windows, Mac, Linux                                             |
| **Open Source & Free**         | Anyone can use and contribute                                           |
